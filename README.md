# Taxi Route Finder

## Overview
A Java-based application that uses Dijkstra’s algorithm to calculate the shortest and most efficient taxi routes between locations.

## Features
- Graph-based route representation
- Shortest path calculation using Dijkstra’s algorithm
- Multiple route node support
- Efficient pathfinding between destinations

## Technologies
- Java
- Graph Theory
- Dijkstra’s Algorithm

## How It Works
1. Locations are represented as nodes
2. Taxi routes are weighted edges
3. Algorithm computes shortest path between start and destination

## How to Run
1. Compile Java files
2. Run main class
3. Input start and end locations
